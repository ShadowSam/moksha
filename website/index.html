What is Moksha?

A Platform

Moksha is a highly-scalable platform for creating dashboards of live widgets. Moksha makes it simple for these widgets to extract and extend data from arbitrary resources in an intuitive & efficient manner.

An Architecture

Moksha brings existing technologies together to create a next-generation web architecture that is designed to simplify the creation of create rich, live web applications that can easily bridge existing services.

A Framework

Compiled from the best-of-breed components, Moksha applications are written using stable pre-existing modules that allow for the rapid creation of powerful applications.

A Hub

Moksha also acts as a live notification hub that allows people and other applications and services to create, publish, and subscribe to arbitrary message streams.

Features

  • Free & Open Source
  • Supported by open standards
  • Comprehensive WSGI middleware stack
  • Application and widget plugin architecture
  • Low-latency browser socket
  • Live widget API
  • Supports 100% AMQP from the browser to the broker
  • Scalable architecture
  • External resource connectors
  • Messaging hub that can integrate with an AMQP broker
  • Message producer/consumer API
  • Leverages the power of TurboGears2
  • Dynamic extension points
  • Dashboard creation API
