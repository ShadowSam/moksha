Widgets
=======

A ``Widget`` in Moksha is simply a `ToscaWidget <http://toscawidgets.org>`_.
ToscaWidgets2 is itself a framework for building reusable web components.  It
is "application-framework" agnostic like Moksha, and so can be used with other
application frameworks like TurboGears2, Pyramid, and Flask.  TurboGears2 ships
Toscawidgets2 out of the box.

Along with their official documentation, you can also find great ToscaWidgets
tutorials in the `TurboGears2 <http://www.turbogears.org/2.2/docs/toc.html>`_
and `ToscaWidgets2 <http://tw2core.rtfd.org>`_ docs.
