===================
Hacking with Moksha
===================

Getting the code
----------------

.. code-block:: bash

    $ git clone git://git.fedorahosted.org/git/moksha

Starting
--------

.. code-block:: bash

    $ ./start-moksha

Stopping
--------

.. code-block:: bash

    $ ./stop-moksha


Generating documentation
------------------------

.. code-block:: bash

    $ paver html

Running the test suite
----------------------

.. code-block:: bash

    $ nosetests
