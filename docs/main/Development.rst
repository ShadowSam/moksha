===================
Hacking with Moksha
===================

Getting the code
----------------

.. code-block:: bash

    $ git clone git://git.fedorahosted.org/git/

Running
-------

.. code-block:: bash

    $ paster serve development.ini

Generating documentation
------------------------

.. code-block:: bash

    $ paver html

Running the test suite
----------------------

.. code-block:: bash

    $ nosetests
