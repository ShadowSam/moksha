===================
Hacking with Moksha
===================

Getting the code
----------------

.. code-block:: none

    git clone git://git.fedorahosted.org/git/

Running
-------

.. code-block:: none

    paster serve development.ini

Generating documentation
-------------------------------

.. code-block:: none

    paver html

Running the test suite
-----------------------------

.. code-block:: none

    nosetests
